# Help Hotlines Website

A simple, free, and open-source website listing **hotlines for mental health, substance use, and other support services**.

This project is designed to be **fast, reliable, and accessible**, with a focus on helping people in need. It is not proprietary software — it’s released under the **MIT License**.

---

## Features

- Country selector (USA, UK, Canada, Australia)
- Click-to-call hotlines
- Section for helping someone else
- Printable layout
- Fully open-source and deployable via GitHub Pages
